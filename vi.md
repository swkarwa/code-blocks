## VI Editor commands

### delete current line
    - it should be not in insert mode
    
```bash
dd
```

## preseve formatting while pasting

```bash
:set paste
```

## delete word after cursor
    - dw

## Deletes from cursor to start of next word
    - dW

## to move line in vi
``` bash
    - shift + V (selects line)
    - > (for right) , < (for left)
```
