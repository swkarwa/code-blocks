# Typescript

- typescript is a super set of JS, which provide additional features
  - datatypes
  - generics
  - arrow function
  - re-decleration is not allowed

- have .ts extendsion
- ts compiler converts compiled code to .js file
- it can be executed wsithout compilation, and execute TS directly
- JS does not have any types
- compile ts code
```code
npm install -g typescript
tsc <file-path>
```
- isntall typescript executor
```code
npm install -g tsx
```
